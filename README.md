# AI Usage Guidelines (Courses and Laboratories)

This repository contains the **course- and laboratory-level guidelines on the use of Artificial Intelligence (AI) tools**. The guidelines outline permitted and prohibited uses of AI, requirements for transparency, and expectations regarding individual intellectual contribution in coursework and laboratory submissions.

The guidelines apply to written assignments, laboratory reports, projects, and presentations, unless otherwise specified by assignment- or lab-specific instructions.

For full details, please refer to:
- `ai_guidelines.md`
